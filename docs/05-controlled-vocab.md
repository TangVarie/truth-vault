# 05 · 受控词表 v0.2

## 为什么存在

Essence 层和 Audience 层的标注必须基于**闭集词表**。LLM 自由表达跨样本不可比，跨项目无法对比。这个文档定义所有受控词表的当前版本。

**词表版本**: v0.2 (Ziao review 后定稿)
**最后更新**: 2026-05-18
**v0.1 → v0.2 变更**: 见文末"版本变更日志"

> 词表是 Truth Vault 的核心约束。词表错了 essence 层就废。v0.2 已通过策略 lead review，可以投入历史数据回标。

---

## 词表演化规则

1. **版本号永远递增**（v0.1 → v0.2 → v1.0）
2. **每条数据记录标注时的词表版本**（`essence_vocab_version` 字段）
3. **添加新值容易，删除现有值困难** —— 删除需要先把所有使用此值的历史数据迁移
4. **每次修改词表必须创建新决策记录**（DECISIONS.md）

## 3. `emotional_lever` · 情绪杠杆（**12 个值** ⭐ v0.2 新增 2 个）

文案触发反应的根本情绪机制。这是 essence 层最核心的字段。

### 负向 5 件套（撬动负面情绪）

| 值 | 定义 | 典型 angle |
|---|---|---|
| `焦虑撬动` | 激发对未来不确定性的**担心**（模糊、长程、未发生） | "再不...就晚了"、"35+ 了还..."、"我会不会..." |
| `羞耻撬动` | 激发自我形象受损感 | "被嫌弃"、"丢人"、"不好意思" |
| `恐惧撬动` | 激发对**具体威胁**的**怕**（具体、即时、有对象） | "医生说我...了"、"她当面说..." |
| `愤怒撬动` | 激发对外部对象的不满 | 对行业、对群体、对现象 |
| `罪恶感撬动` ⭐ | 激发"做得不够好"的愧疚 | "为孩子/父母这么做都不愿意"、"我对不起..." |

### 正向 5 件套（建立正面情绪）

| 值 | 定义 | 典型 angle |
|---|---|---|
| `造梦投射` | 描绘理想生活/未来 | "成为...的样子"、"过上...的生活" |
| `认同感建立` | "你不是一个人"的共鸣 | "懂的都懂"、"姐妹们都这样" |
| `归属感建立` | 建立群体身份认同 | "我们 35+ 女性"、"宝妈群体" |
| `共鸣释放` | 表达被压抑的情绪 | "终于敢说"、"憋了好久" |
| `虚荣撬动` ⭐ | 让读者感到优于某个对照群体 | "成功人士都用..."、"懂行的姐妹才知道..." |

### 中性 2 件套

| 值 | 定义 | 典型 angle |
|---|---|---|
| `好奇驱动` | 激发探索欲 | "竟然"、"原来"、"没想到" |
| `信息差利用` | 提供独家/稀缺信息 | "内部消息"、"行业内幕"、"少有人知" |

### 焦虑 vs 恐惧 边界规则 ⚠️ 重要

这两个最容易混淆。判断规则：

- **焦虑** = 偏"担心"：对未来不确定性的情绪，**模糊、长程、未发生**
- **恐惧** = 偏"怕"：对具体威胁的反应，**具体、即时、有对象**

判断步骤：
1. 文案是否描述了**具体已发生的威胁/事件**？
   - 是 → 倾向恐惧
   - 否 → 倾向焦虑
2. 威胁对象是否**具体可指**？（具体人、具体诊断、具体场景）
   - 是 → 恐惧
   - 否 → 焦虑

对照例子：

| 文案 | 判定 | 理由 |
|---|---|---|
| "再不戒烟就影响下一代了" | **焦虑** | 模糊未来"影响下一代"，未发生 |
| "医生说我肝硬化早期了" | **恐惧** | 具体诊断，已发生 |
| "55+ 还找不到对象怎么办" | **焦虑** | 模糊未来，开放式 |
| "她当面说我又老又油腻" | **恐惧** | 具体事件，已发生 |
| "万一以后..." | **焦虑** | 关键词"万一""以后" |
| "刚才医生告诉我..." | **恐惧** | 关键词"刚才"+具体对象 |

### 虚荣撬动 vs 造梦投射 边界

也容易混淆。差别在于：
- **造梦投射** = 给读者描绘**他想成为的样子**（aspiration，未来）
- **虚荣撬动** = 让读者**已经感到**自己优于某个对照群体（superiority，现在）

例：
- "用了这个，成为别人羡慕的精致女孩" → **造梦投射**（aspiration）
- "懂行的姐妹才用这个，别买错了" → **虚荣撬动**（让懂的人感到优越）

### 罪恶感撬动 vs 焦虑撬动 边界

也容易混淆。差别在于：
- **焦虑撬动** = 担心**未来不好**
- **罪恶感撬动** = 感到**现在做得不够**

例：
- "再不给孩子用最好的，将来后悔" → **焦虑撬动**（未来后悔）
- "别的妈妈都给孩子用这个，我居然没有" → **罪恶感撬动**（现在愧疚）

## 7. `trend_dependencies` · 时效依赖（**10 个值** ⭐ v0.2 关键重构）

标识这条文案依赖哪些时效性元素。**用于判断时间衰减速度**。

### v0.2 关键改动：从二级时间分层 → 三级时间分层

v0.1 把所有"当代语言相关"压缩到一个"当代流行语"标签。v0.2 拆分为两层，加入**时代语言范式**作为新的中间层：

```
通用 (perpetual)              ← 半衰期 5 年+，几乎不衰减
   ↑
时代语言范式 (era_idiom)      ← 半衰期 2-3 年，反映时代特征
   ↑
当代流行词 (current_buzzwords) ← 半衰期 6-12 个月，快速过期
```

**为什么这个分层重要**：在"绝绝子"这种具体流行词和"完全不依赖时代"的纯通用之间，存在**结构性话术模式**。识别这些模式是"引领新话术而不是模仿"的基础。当当代流行词过期时，依然可以基于范式造下一个词。

### 完整 10 个值

| 值 | 半衰期 | 定义 |
|---|---|---|
| `特定平台事件` | 短 | 引用具体小红书/抖音热点事件 |
| `特定IP引用` | 短-中 | 提到具体明星、综艺、电影、电视剧、网红 |
| `时事热点` | 短 | 引用当前新闻、社会事件 |
| `季节性事件` | 短 | 双 11、春节、夏季等 |
| `节日` | 短 | 节假日相关 |
| `行业事件` | 短-中 | 品类内特定事件（如美妆周、戒烟日） |
| `当代流行词` | 6-12 月 | 具体的流行词汇（"绝绝子"、"yyds"、"真的会谢"、"米色裤子"）|
| `时代语言范式` ⭐ | 2-3 年 | 当代特有的**结构性话术模式**，比具体词更持久 |
| `平台话术` | 中 | 强依赖某平台特定语言（小红书"姐妹们"等基础话术） |
| `通用` | 5 年+ | 不依赖任何时效性元素，可穿越周期（排他：含通用则不含其他） |

### 时代语言范式 的具体子模式（参考用，不进入闭集）

为方便标注理解，"时代语言范式" 的常见子类型：

- **夸张式自嘲**：把自己描述得很惨/很失败以引发共鸣（"我废了"、"摆烂中"）
- **反向表达**：用反义词或否定句表达肯定（"真的会谢"=真的不行）
- **缩写+情绪化**：高密度信息压缩 + 强情绪（"yyds"、"绝绝子" 是这个范式的实例化）
- **emoji 配文化**：emoji 作为语义/情绪载体而非装饰
- **数字+夸张**："99% 的人都不知道"、"血压拉满"
- **拟人化语言**：把抽象事物当人讲（"我的卡饱了"、"焦虑在我胃里跳广场舞"）

标注时**只标"时代语言范式"这个总标签**，不需要细分到子模式。子模式在 reasoning 字段里描述即可。

### "通用"的严格定义（v0.2 校准）

"通用"标签的判断要严格：
- 文案中**没有任何当代流行词**
- 文案中**没有任何时代语言范式**
- 不依赖平台话术（不含"姐妹们"等小红书特征语言）
- 不引用任何具体 IP / 事件 / 时事

**通用样本是稀有但高价值的**——这种文案的内核可以迁移到任何时代、任何平台。

例子：
- "走在路上突然觉得，我已经很久没有为自己做过一件事了" → **通用**（内核：自我觉察，无时代依赖）
- "姐妹们，谁懂这种感觉" → **平台话术** + 时代语言范式
- "yyds 这个产品真的绝" → **当代流行词** + 时代语言范式

## 9. `inferred_audience_profile` · 受众画像（JSONB 结构）

不是闭集字段，是结构化 JSONB。结构如下：

```yaml
inferred_audience_profile:
  # ── 闭集部分（统计可比）──
  demographic:
    age_band: ["30-39", "40-49"]      # 多选，受控值 [20-29, 30-39, 40-49, 50+]
    gender_skew: female               # female / male / mixed
    city_tier: ["1线", "新1线"]        # 多选，受控值 [1线, 新1线, 2线, 3-4线, 5线及以下]
    life_stage: "育儿期"               # 闭集: 学生 / 职场新人 / 已婚未育 / 育儿期 / 空巢 / 退休
    value_orientation: "精致"          # 闭集: 务实 / 精致 / 自洽 / 表达 / 反叛
    income_band: "中产"                # 闭集: 学生 / 入门 / 中产 / 高净值

  # ── 自由文本部分（信息密度优先）──
  psychographic:
    primary_pain: "孩子教育焦虑 + 自我职场焦虑"
    primary_aspiration: "希望成为有品味、不被定义的中年女性"
    likely_objections: "价格、是否真的有效"

  # ── 元数据 ──
  confidence: 0.7                      # LLM 推断置信度（自评 0-1）
  inferred_at: "2026-05-18T12:00:00"
```

## 时间衰减权重的工程实现（v0.2 三级分层）

检索 / 训练时，每条历史样本的权重按层独立衰减。v0.2 引入三级时间分层后的实现：

```python
def sample_weight(sample, query_date):
    age_months = (query_date - sample.publish_date) / 30

    # Surface 层的时间衰减按 trend_dependencies 精细化
    deps = set(sample.trend_dependencies)

    if "通用" in deps:
        surface_halflife = 60   # 5 年
    elif "时代语言范式" in deps and not (deps & SHORT_TERM_DEPS):
        # 有范式标签但没有短期标签（流行词/IP/事件...）
        surface_halflife = 30   # 2.5 年
    elif "当代流行词" in deps or (deps & SHORT_TERM_DEPS):
        surface_halflife = 9    # 9 个月
    else:
        surface_halflife = 12   # 默认 1 年

    surface_weight = 0.5 ** (age_months / surface_halflife)
    essence_weight = 0.5 ** (age_months / 60)   # essence 半衰期 5 年
    audience_weight = 0.5 ** (age_months / 30)  # audience 半衰期 2.5 年

    return {
        'surface': surface_weight,
        'essence': essence_weight,
        'audience': audience_weight,
    }

SHORT_TERM_DEPS = {
    "特定平台事件", "特定IP引用", "时事热点",
    "季节性事件", "节日", "当代流行词"
}
```

意义：3 年前的"绝绝子型"内容 surface 权重 ≈ 0.1（几乎无用），但 3 年前的"通用型"内容 surface 权重 ≈ 0.7（依然有效）。**这就是"引领新话术"的算法基础**——识别哪些老样本的话术结构依然有效，提取范式而不是抄词。
